# Demo Pitches

**Time:** 15:45-16:05

## Purpose
Make the results visible and compare the value of the different feature extensions.

## Key Question
What did we build, what was easier than expected, and what remains difficult?

## Pitch Structure
- Initial idea.
- What we used from Nanobot.
- What we changed, configured, documented, or tested.
- Live result or fallback evidence.
- Main limitation.
- One next step.
- One learning about another team's perspective.

## Timebox
- Maximum four minutes per group.
- Prefer a working slice over a broad explanation.

## Audience Questions
- What would a visitor understand from this demo?
- What would an operator need to keep this running?
- What should be reused in future demonstrators?

## Expected Output
- Concise track demos.
- Shared comparison of technical and communication value.
