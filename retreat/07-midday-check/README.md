# Midday Check

**Time:** 12:55-13:00

## Purpose
Create a shared view of what works, what is blocked, and what should/will happen after lunch.

## Key Question
What works, what not, and what should be addressed after lunch? Are there already some ideas; Feedback?

## Format
- Each working group contributes one status point.
- Each working group contributes one next step.
- Facilitator identifies which open points belong to feature tracks, deployment, or documentation.

## Expected Output
- Short status list in the shared notes or cheatsheet.
